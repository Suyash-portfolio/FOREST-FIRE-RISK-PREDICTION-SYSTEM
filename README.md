# 🌲 Forest Fire Risk Prediction System

## 📌 Overview

Forest fires are dangerous natural disasters that can cause severe damage to the environment, wildlife, and human life.
This project uses **Machine Learning** to predict whether a forest fire is likely to occur based on environmental and geographical data.

The system helps in **early detection and prevention**, allowing better planning and resource management.

---

## 🎯 Objective

* Predict forest fire occurrence using environmental data
* Identify high-risk areas
* Visualize fire-prone regions using maps
* Support disaster management decisions

---

## 📊 Dataset Features

The dataset includes the following features:

* X, Y → Spatial coordinates (location)
* FFMC → Fine Fuel Moisture Code
* DMC → Duff Moisture Code
* DC → Drought Code
* ISI → Initial Spread Index
* Temp → Temperature
* RH → Relative Humidity
* Wind → Wind speed
* Rain → Rain amount
* y → Fire occurrence (0 = No Fire, 1 = Fire)

👉 As shown in your dataset slide (page 5), these features represent environmental conditions affecting fire risk
